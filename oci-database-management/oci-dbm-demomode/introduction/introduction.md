# Introduction

## About this Workshop

In this workshop, you will explore Oracle Cloud Infrastructure Database Management using Demo Mode. Demo Mode lets you review key Database Management capabilities with preconfigured demo data, so you can learn the service flow without setting up, connecting, or managing your own databases.

This workshop is designed as a guided, read-only experience. You will enable Demo Mode, open the Database Management pages, review fleet-level information, and drill into database monitoring and performance views using the demo environment.

Estimated Workshop Time: 1 hour

### About Oracle Cloud Infrastructure Database Management

Oracle Cloud Infrastructure Database Management provides a single pane of glass to monitor and manage Oracle Databases across cloud and on-premises environments. Database Management helps database administrators, DevOps users, and operations teams monitor fleet health, review performance and configuration metrics, investigate database activity, and explore diagnostic information.

Database Management supports Oracle Databases located on-premises and connected through Oracle Cloud Infrastructure External Database, Oracle Autonomous Databases, Oracle Databases in third-party cloud environments, and Oracle Cloud Databases on co-managed Oracle Database Cloud solutions such as:

- Bare Metal and Virtual Machine DB Systems
- Exadata Cloud Service

Using Database Management, you can:

- Monitor key performance and configuration metrics across a fleet of Oracle Databases.
- Review fleet health, inventory, resource usage, alarms, and database status.
- Drill into managed database details for additional monitoring information.
- Use Performance Hub for database performance analysis.
- Review diagnostic information such as SQL Monitoring, ADDM, AWR Explorer, and related performance views where available in the demo environment.

### Objectives

In this workshop, you will:

- Enable Database Management Demo Mode.
- Navigate to the Database Management landing page.
- Explore fleet-level monitoring information using demo data.
- Review managed database details and monitoring charts.
- Explore performance and diagnostic views available in Demo Mode. Performance Hub, AWR Explorer, and ADDM Spotlight use demo-environment data where available.
- Learn how Database Management helps users understand database health and performance without creating or modifying resources.

### Prerequisites

- Access to an Oracle Cloud Account or LiveLabs environment that supports Database Management Demo Mode.
- Access to the required policies for viewing Database Management Demo Mode. If the policies are missing, the Demo Mode policy panel can guide an administrator through the required read-only endorse policy.
- No database setup is required for this workshop.

**Note:** This workshop uses Demo Mode and demo data. The steps are intended for exploration and learning. Do not create, update, or delete resources as part of this workshop. Some administration and write workflows are hidden or disabled while Demo Mode is active.
